<ChallengeTitle>Anchor Escrow</ChallengeTitle>

Sekarang saatnya untuk menguji apa yang telah Anda pelajari. Dengan menggunakan apa yang telah Anda pelajari dari pelajaran ini, buatlah program escrow menggunakan framework Anchor yang memenuhi persyaratan berikut:

> Akun state Anda harus didefinisikan menggunakan diskriminator akun dengan nilai `1`.

<RequirementList>
  <Requirement title="Tantangan 1: Membuat penawaran escrow">
    Program Anda harus memungkinkan pembuat untuk membuat catatan escrow dengan persyaratan pertukaran. Definisikan instruksi untuk melakukan ini menggunakan diskriminator instruksi dengan nilai `0`.
  </Requirement>

<Requirement title="Tantangan 2: Mengambil penawaran escrow">
  Program Anda harus memungkinkan pengambil untuk menerima persyaratan
  pertukaran escrow. Definisikan instruksi untuk melakukan ini menggunakan nilai
  diskriminator instruksi dengan nilai `1`.
</Requirement>

  <Requirement title="Tantangan 3: Membatalkan penawaran escrow">
    Program Anda harus memungkinkan pembuat untuk membatalkan penawaran escrow dan mengembalikan dana yang terdapat dalam akun escrow. Pastikan instruksi Anda didefinisikan dengan nilai diskriminator instruksi dengan nilai `2`.
  </Requirement>
</RequirementList>
